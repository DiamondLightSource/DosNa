Dosna Webapp

How to use

To view objects inside a pool use /list/obj/<poolname>
For example /list/obj/test-dosna

To display image of the object use /display/img/<pool>/<obj>
For example display/img/test-dosna/190903132342_data_

To display a particular slice of an object use /display/img/<pool>/<obj>/slice/< xslice >/< yslice >/< zslice >
For example display/img/test-dosna/190903132342_data_/slice/0:225/0:225/0:43
